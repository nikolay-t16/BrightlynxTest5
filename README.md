# brightlynxtest5 
делится на две части проекта front - vue.js (cli) и server - node.js (koa)

##Для запуска front

```
cd front
npm install
npm run serve
```

##Для запуска server

```
cd server
npm install
node server.js
```

В качестве базы использовал mysql запущенный на open server
